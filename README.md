PCROOM
======

各程式功能
---------
* CSCC_INSTALL
    * 整合所有win7安裝的過程
    * 執行 ".\cs-setup.ps1 all" 即可跑完所有流程
    
* CSCC_sidchg.bat
    * 使用前請先填入key
    * 點兩下後會開始跑sid, 跑完後會自動重啟電腦

* CSCC_Wallpaper/*
    * CSCC.psd 為原圖
    * CSCC.png 為一般圖檔
    * 原圖: pthsieh
    
* 316_BIOS.zip
    * 316 原先電腦 BIOS有 OEM 限制
    * BIOS_Acer_P01.B0_Windows/WinTool/64/WFlash2M.BAT
